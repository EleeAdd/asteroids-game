# 🚀 Planetarium Interactive Game (Team Project)

This project is an interactive browser-based game developed as part of my internship at the P-Tech Planetarium in collaboration with a team.

The game is designed for a touchscreen environment where users interact with space-themed elements such as enemies, rockets, and scoring mechanics.

---

## 🧑‍💻 My Contributions

I focused on gameplay logic, interactivity, and system behavior:

- Implemented the **scoring system**, including real-time score updates and high score logic
- Built **collision detection logic** between rockets and enemies
- Developed **rocket targeting system** (click-based enemy targeting and movement)
- Fixed issue where multiple rockets incorrectly increased score (ensured score updates only on first hit)
- Improved **game responsiveness and user interaction**
- Worked on **UI elements** such as score display and animations (explosions, effects)
- Assisted in debugging and refining gameplay behavior

---

## 🎮 Features

- Interactive click-to-shoot gameplay
- Dynamic enemy spawning and movement
- Collision detection system
- Real-time scoring system
- Visual effects (explosions, animations)
- Designed for touchscreen interaction
